## Exercises (make you job-ready)

Complete these in order. Each task teaches a real skill.

### Level 1 (easy)
1) Add a new KPI card to the dashboard.
2) Add a new field to the patient form (example: "preferredLanguage").
3) Show that field in the patients table.

### Level 2 (medium)
1) Add a new permission: `reports.read`.
2) Create a `/reports` page that requires that permission.
3) Show a simple table with fake data.

### Level 3 (advanced)
1) Add a new model `Clinic`.
2) Link Providers to a Clinic.
3) Add a clinic selector on the Providers page.

### How to check your work
- Run `npm run lint`
- Run `npm run test`
- Run `npm run dev` and click through the UI
